
# Project Quick Setup and Start

1. Install Anaconda-3.5.2  by adding environment variables(check the box) during setup
2. Install VS Code-1.25.1 for IDE.
3. Install virtualenv 'pip install virtualenv'
4. Create a project folder 'mkdir Vaagdevi'
5. Navigate to Project folder ' cd Vaagdevi'
6. Clone git repo 'git clone https://github.com/sahitesh/student_attendance'
7. Create a virtual environment for the project 'virtualenv vaagdevi_env'
8. Activate the environment 'vaagdevi_env\Scripts\activate'
9. Navigate to student_attendance\attendanceLedger 'cd student_attendance\attendanceLedger'
10. Install requirements 'pip install -r requirements'
11. Install the whl file for mysqlclient 'pip install mysqlclient-1.3.13-cp36-cp36m-win_amd64.whl'
12. Start the server 'python manage.py runserver'

# Project Quick Launch

1. VS Code starts in git repo 'student_attendance'
2. Go back a directory('cd..') and activate environment 'vaagdevi_env\Scripts\activate'
3. Navigate to Python project directory 'cd student_attendance\attendanceLedger'
4. Start the server 'python manage.py' runserver
